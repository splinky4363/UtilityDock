# ToolHub

A modern collection of free online tools, calculators, converters, and generators.

## Features
- Fast and lightweight
- Mobile-friendly design
- No login required
- Free to use
- Includes calculators, converters, text tools, and generators

## Categories
- Calculators (VAT, percentage, BMI, etc.)
- Converters (currency, units, time, etc.)
- Generators (passwords, QR codes, usernames, etc.)
- Text utilities (word counter, case converter, etc.)

## Purpose
This project aims to provide simple and useful tools for everyday tasks, all in one place.

## Tech Stack
- HTML
- CSS
- JavaScript
- GitHub Pages
