About this tool

Generate one or many random whole numbers within a range you define, using the browser's cryptographically secure random number source for unbiased results — handy for raffles, sampling, games or picking a winner.

How to use the random number generator

  1. Set a minimum and maximum value.
  2. Choose how many numbers you want and whether duplicates are allowed.
  3. Click “Generate” to get your numbers, then copy them.

Frequently asked questions

Can I generate numbers without repeats?

Yes — turn off “Allow duplicates” and each generated number will be unique, as long as the range is large enough to fit the quantity requested.

What's the maximum range?

Any integer range your browser can represent — practically, up to several billion — though very large unique sets take longer to compute.

Is the randomness good enough for a giveaway?

It uses the Web Crypto API, which is suitable for fair, unbiased selection in casual giveaways and games.
