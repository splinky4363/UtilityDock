About this tool

This generator builds passwords using your browser's cryptographically secure random number generator (crypto.getRandomValues), not Math.random, so results are suitable for real account security.

Nothing is transmitted anywhere — the password is created and shown entirely on your device.

How to use the password generator

  1. Drag the length slider to the size you want.
  2. Toggle which character sets to include.
  3. Click “Generate” (or change any option) to get a new password, then copy it.

Frequently asked questions

Is this password generator secure?

Yes. It uses the Web Crypto API's getRandomValues() for randomness, the same source browsers use for cryptography, and runs 100% client-side.

What makes a password strong?

Length matters most. Aim for at least 16 characters mixing upper and lower case letters, numbers and symbols, and never reuse passwords across sites.

Should I save my generated password somewhere?

Use a password manager to store it. This tool doesn't save or remember anything you generate.
