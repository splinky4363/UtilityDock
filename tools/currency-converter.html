About this tool

This is a static, offline currency converter: exchange rates are stored as editable numbers in the page's JavaScript rather than fetched from a live API. That keeps it fast, free of external dependencies, and fully functional on GitHub Pages with no backend.

Because the rates are fixed demo values, always check a live source before making real financial decisions.

How to use the currency converter

  1. Enter an amount and pick the currency you're converting from and to.
  2. The converted amount updates instantly using the built-in demo rates.
  3. Open “Edit demo rates” to tweak any rate and see the conversion recalculate live.

Frequently asked questions

Are these live exchange rates?

No. They are illustrative demo rates hard-coded in JavaScript so the tool works instantly without calling an external API. Edit them yourself in the “Edit demo rates” panel if you know current rates.

Why not use a live rates API?

UtilityDock is a fully static site with no server. A live API would require a backend or a third-party key, add latency, and could fail offline — a transparent editable demo keeps the tool fast and dependency-free.

Can I add more currencies?

The rate table is a plain JavaScript object, so a developer can add another currency code and rate in a couple of lines.
