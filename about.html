About UtilityDock

Free tools, built to be fast.

UtilityDock is a small, focused collection of everyday tools — calculators, converters, generators and text utilities — built as a single fast, static website with no accounts, no paywalls and no tracking scripts.

Every tool runs entirely in your browser using plain HTML, CSS and JavaScript. There's no backend server processing your data: when you calculate a BMI, generate a password or count words, the computation happens on your device and nothing is uploaded anywhere.

Why we built it

Most "free tool" sites online are slow, cluttered with ads, or require sign-up for basic features. We wanted the opposite: open a page, use the tool, get your answer — in well under a second.

Fast

No frameworks, no build step, minimal DOM — pages are lightweight and load instantly, even on slow connections.

Private

No analytics, no cookies, no accounts. Your inputs never leave your browser.

Free

Every tool is free to use, with no limits, sign-up walls or premium tiers.

Open

Built with plain HTML, CSS and vanilla JavaScript — easy to read, extend and self-host.

What's inside

UtilityDock currently includes 10 tools across 4 categories: Calculators, Converters, Generators, Text Tools. New tools are added by following the same simple page structure used throughout the site.

Questions or feedback?

This is an independent, open project. If you spot a bug or have an idea for a new tool, feel free to open an issue or pull request on the project's GitHub repository.
